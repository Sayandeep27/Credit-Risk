# Expected Credit Loss (ECL) in Credit Risk

---

# 1. What is Credit Risk?

**Credit Risk** is the risk that a borrower will not repay a loan or meet their financial obligations.

In simple words:

> Credit Risk = Risk that the borrower will default (not pay the money back).

### Example

A bank gives a loan of **₹1,00,000** to a customer.

If the customer fails to repay the loan, the bank faces **credit risk**.

---

# 2. What is Expected Credit Loss (ECL)?

**Expected Credit Loss (ECL)** is the **expected amount of money a bank might lose if a borrower defaults.**

Instead of waiting for the default to happen, banks **estimate the loss in advance**.

This concept is used in **IFRS 9 accounting standards**.

### Simple Definition

> Expected Credit Loss = Estimated loss from a loan due to possible default.

Banks calculate this loss using three main components:

1. Probability of Default (PD)
2. Loss Given Default (LGD)
3. Exposure at Default (EAD)

---

# 3. Key Components of Expected Credit Loss

| Component | Full Form              | Meaning                                               |
| --------- | ---------------------- | ----------------------------------------------------- |
| PD        | Probability of Default | Chance that the borrower will default                 |
| LGD       | Loss Given Default     | Percentage of money the bank loses if default happens |
| EAD       | Exposure at Default    | Total loan amount exposed at the time of default      |

These three components are used to calculate **Expected Credit Loss**.

---

# 4. Probability of Default (PD)

## Definition

**Probability of Default (PD)** is the **probability that a borrower will fail to repay the loan within a given time period.**

It is usually measured over **1 year**.

### Example

If a bank estimates:

PD = **5%**

This means:

> There is a **5% chance that the borrower will default within a year.**

### Example Table

| Borrower   | PD  | Meaning               |
| ---------- | --- | --------------------- |
| Borrower A | 2%  | Very low default risk |
| Borrower B | 5%  | Moderate risk         |
| Borrower C | 15% | High default risk     |

Banks estimate PD using:

* Credit scores
* Historical default data
* Financial statements
* Machine learning models

---

# 5. Loss Given Default (LGD)

## Definition

**Loss Given Default (LGD)** is the **percentage of the loan that the bank loses if the borrower defaults.**

Even if a borrower defaults, the bank may recover some money through:

* Collateral
* Asset seizure
* Legal recovery

LGD measures the **remaining loss after recovery**.

### Formula

LGD = (Loss after recovery / Total loan amount)

### Example

Loan amount = **₹1,00,000**

After default, bank recovers = **₹40,000**

Loss = **₹60,000**

LGD = 60%

### Table Example

| Loan   | Loan Amount | Recovery  | Loss    | LGD |
| ------ | ----------- | --------- | ------- | --- |
| Loan A | ₹1,00,000   | ₹40,000   | ₹60,000 | 60% |
| Loan B | ₹2,00,000   | ₹1,50,000 | ₹50,000 | 25% |

---

# 6. Exposure at Default (EAD)

## Definition

**Exposure at Default (EAD)** is the **total amount of money the bank is exposed to when the borrower defaults.**

It represents the **outstanding loan amount at the time of default**.

### Example

Original loan = **₹5,00,000**

Borrower repaid = **₹2,00,000**

Remaining loan = **₹3,00,000**

If default happens now:

EAD = **₹3,00,000**

### Example Table

| Loan   | Original Loan | Amount Repaid | Remaining Loan (EAD) |
| ------ | ------------- | ------------- | -------------------- |
| Loan A | ₹5,00,000     | ₹2,00,000     | ₹3,00,000            |
| Loan B | ₹10,00,000    | ₹6,00,000     | ₹4,00,000            |

---

# 7. Expected Credit Loss Formula

The basic formula used in credit risk models is:

```
ECL = PD × LGD × EAD
```

Where:

PD = Probability of Default
LGD = Loss Given Default
EAD = Exposure at Default

---

# 8. Example of Expected Credit Loss Calculation

Suppose a bank gives a loan with the following parameters.

| Parameter | Value     |
| --------- | --------- |
| PD        | 5%        |
| LGD       | 60%       |
| EAD       | ₹1,00,000 |

Convert percentages to decimals:

PD = 0.05
LGD = 0.60

### Calculation

```
ECL = PD × LGD × EAD

ECL = 0.05 × 0.60 × 100000

ECL = 3000
```

### Result

Expected Credit Loss = **₹3,000**

### Interpretation

> On average, the bank expects to lose ₹3,000 from this loan due to possible default.

---

# 9. Example with Multiple Loans

| Loan   | PD  | LGD | EAD       | Expected Loss |
| ------ | --- | --- | --------- | ------------- |
| Loan A | 2%  | 40% | ₹1,00,000 | ₹800          |
| Loan B | 5%  | 60% | ₹2,00,000 | ₹6,000        |
| Loan C | 10% | 70% | ₹50,000   | ₹3,500        |

Total Expected Loss:

```
800 + 6000 + 3500 = ₹10,300
```

The bank should keep **₹10,300 as provision** for expected credit losses.

---

# 10. Why Banks Calculate ECL

Banks calculate Expected Credit Loss for several reasons:

### 1. Risk Management

Helps banks understand potential losses.

### 2. Capital Planning

Banks keep reserves to absorb losses.

### 3. Regulatory Compliance

Required under **IFRS 9** and **Basel regulations**.

### 4. Better Lending Decisions

Helps banks price loans correctly.

---

# 11. Types of ECL in IFRS 9

Under IFRS 9, loans are divided into **three stages**.

| Stage   | Risk Level                   | Loss Calculation |
| ------- | ---------------------------- | ---------------- |
| Stage 1 | Normal loans                 | 12‑month ECL     |
| Stage 2 | Significant increase in risk | Lifetime ECL     |
| Stage 3 | Defaulted loans              | Lifetime ECL     |

### 12‑Month ECL

Loss expected in the **next 12 months**.

### Lifetime ECL

Loss expected **over the entire life of the loan**.

# 12. Simple Real‑World Example

A bank gives a **home loan of ₹20,00,000**.

Risk model estimates:

PD = 3%
LGD = 25%
EAD = ₹20,00,000

### Calculation

```
ECL = 0.03 × 0.25 × 2000000

ECL = 15000
```

### Result

Expected Credit Loss = **₹15,000**

Meaning:

> The bank expects an average loss of ₹15,000 from this loan due to default risk.

| PD          | Probability borrower will default      |
| LGD         | Percentage of loss if default occurs   |
| EAD         | Outstanding loan amount at default     |
| ECL         | Expected loss from a loan              |

### Main Formula

```

```

---

# 14. Summary

Expected Credit Loss is a **forward‑looking credit risk measure** used by banks to estimate potential loan losses.

It is calculated using three core components:

* **Probability of Default (PD)**
* **Loss Given Default (LGD)**
* **Exposure at Default (EAD)**

By estimating ECL, banks can:

* Manage risk better
* Maintain financial stability
* Follow regulatory standards
